# 変更履歴 (CHANGELOG)

PTSIアプリケーションの変更履歴を記録します。

## [2.2.0] - 2025-09-25

### 🚀 統計表示とダッシュボードの大幅強化

#### 📊 高度な統計表示機能
- **リアルタイム統計カード**
  - 今日の予約数、登録学生数、総予約数、出席率の表示
  - トレンド表示（前期比較とトレンドアイコン）
  - カラーコーディングによる視覚的フィードバック

- **クラス別統計**
  - 各クラス（ひまわり組、ばら組、さくら組、すみれ組）の詳細統計
  - 在籍数、今日の出席数、出席率の一覧表示
  - レスポンシブグリッドレイアウト

#### 📈 Chart.js統合による高度なデータ可視化
- **月次予約トレンドチャート**
  - 折れ線グラフによる予約数の推移表示
  - エリアチャートによる視覚的強調
  - インタラクティブなホバー効果

- **週次出席率チャート**
  - 棒グラフによる曜日別出席パターン表示
  - カラフルな色分けによる識別性向上
  - パーセンテージ表示機能

- **クラス別分布チャート**
  - ドーナツチャートによるクラス構成の可視化
  - カスタムカラーパレット
  - 凡例付きの詳細表示

#### 🔔 リアルタイム監視システム
- **システムアラート機能**
  - 重要度別アラート表示（警告/情報/エラー/成功）
  - アイコンとカラーによる直感的な識別
  - タイムスタンプ付きアラート履歴

- **最近の活動監視**
  - 予約、ログイン、チャット、システム活動の履歴表示
  - アイコン付きの活動タイプ識別
  - リアルタイム更新機能

#### 📋 包括的な分析・レポート機能
- **期間選択分析**
  - 週次/月次/年次でのデータ期間切り替え
  - 期間に応じた動的データ更新
  - インタラクティブな期間選択ボタン

- **多形式レポートエクスポート**
  - PDF形式でのレポート出力
  - Excel形式でのデータエクスポート
  - CSV形式での詳細データ出力
  - 進捗表示付きエクスポート機能

#### 🎨 モダンなUI/UXデザイン
- **レスポンシブデザイン対応**
  - モバイル・タブレット・デスクトップ対応
  - 適応的グリッドレイアウト
  - フレキシブルなコンポーネント配置

- **アニメーション効果**
  - スムーズな画面遷移
  - ホバーエフェクト
  - フェードイン/スライドインアニメーション
  - チャートアニメーション

- **通知システム**
  - 操作結果のリアルタイム通知
  - 成功/エラー/情報/警告の通知タイプ
  - 自動消去機能付き通知

#### 🔧 技術的改善
- **フロントエンド強化**
  - Chart.js 4.x統合による高性能データ可視化
  - CSS Grid/Flexboxによるモダンレイアウト

- **データ管理機能**

  - エラーハンドリング強化
  - CSVデータとの統合

#### 📁 変更されたファイル

## [2.1.0] - 2025-09-25

### 🆕 新機能追加

#### 職員間グループチャット機能
- **新しいファイル**
  - `staff-chat/staff-chat.html` - 職員チャット画面
  - `staff-chat/staff-chat.js` - 職員チャット機能
  - `staff-chat/staff-chat.css` - 職員チャット専用スタイル

- **機能詳細**
  - 5つの専用チャットルーム
    - 全職員チャット (`staff-general`)
    - 担任教師チャット (`staff-teachers`)
    - 管理職チャット (`staff-admin`)
    - サポートスタッフチャット (`staff-support`)
    - 緊急連絡チャット (`staff-emergency`)
  
  - 緊急連絡機能
    - 緊急送信ボタン - 優先度の高いメッセージ
    - 全職員ブロードキャスト - 全ルームに一斉送信
    - 視覚的な緊急表示 - 赤色強調とアニメーション
  
  - オンライン職員表示
    - リアルタイム職員数表示
    - オンライン職員一覧モーダル
    - 部署別職員情報

#### 操作マニュアル機能
- **新しいファイル**
  - `manuals/parent-manual.html` - 保護者向け操作マニュアル
  - `manuals/admin-manual.html` - 運営者向け操作マニュアル

- **機能詳細**
  - ブラウザ印刷機能でPDF生成対応
  - 美しいグラデーションデザイン
  - 印刷専用ヘッダーとボタン
  - レスポンシブ対応

### 🔧 既存機能強化

#### サーバー側 (server.js)
- **新しいAPIエンドポイント**
  - `GET /staff-chat` - 職員チャットページ
  - `POST /staff-chat/send` - 職員メッセージ送信
  - `GET /staff-chat/messages` - 職員メッセージ取得
  - `POST /staff-chat/broadcast` - 緊急ブロードキャスト
  - `POST /api/admin-info` - 管理者情報取得
  - `GET /download/parent-manual` - 保護者マニュアル
  - `GET /download/admin-manual` - 運営者マニュアル

- **データ構造拡張**
  - 職員チャット用メッセージ格納領域追加
  - 管理者情報にdepartment項目追加
  - 職員2名追加（栄養士、事務職員）

#### 管理者ダッシュボード
- **ファイル変更**: `admin-dashboard/admin-dashboard.html`, `admin-dashboard/admin-dashboard.js`
- **機能追加**
  - 「職員チャット」メニュー項目追加
  - 職員チャットページへの遷移機能

#### ログインページ
- **ファイル変更**: `login/login.html`, `admin-login/admin-login.html`
